# Web_Dev_Quiz_Page
A quiz page with a php backend   html  , CSS , JavaScript as the frontend.
 
This project will help you understand key web development principles, including dynamic content loading, client-side state management, and real-time user interaction using PHP, MySQL, and JavaScript.

Arabic Competence Exam
Key Features and Implementation:
a.	Home Page:
o	Develop a home page that displays general information about the Exam such as duration, levels, number and type of questions, and the result of the exam. It should also provide a clarification about levels of competence. This page provides information about the Arabic language skills of reading and understanding, listening and understanding, composition, grammar and orthographic rules, and speaking. Other information should also be provided, such as information about registration for new users, and login for registered users. This list can be static or dynamically retrieved from a MySQL database using PHP.
b.	Processes:
o	Implement functionality to fetch questions distributed into different levels from the MySQL database using PHP.
o	Design a general page that is populated with the question text, display the choices with only 1 correct answer, and enable the user to submit, rest and change the choice.
o	Use JavaScript to manage the state of the exam on the client-side. Enable users to navigate through all questions (change answers if needed) before submitting the user’s answers to the questions. The JavaScript should track the progress through the exam.
c.	Reports and Feedback:
o	Develop a mechanism to calculate the user's score and display feedback immediately upon level completion using JavaScript. Evaluate the user’s performance by giving an overall score, rating using stars, or any other evaluation presentation.
o	Develop the necessary reports for your projects. The reports must enable users to query about data stored in the database. The report should display the retrieved data in a well-formatted display.


Note: The style and presentation of the site will be considered when evaluating the project.
